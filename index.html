Naとその化合物

1.Na

・製法

＿＿＿＿ 溶融塩電解(融解塩電解)
塩化ナトリウム・水酸化ナトリウムの固体をそのまま融解した液体を電気分解する。 　　　[陰極] ＿＿＿＿ Na⁺+e⁻→Na

・性質

① 密度が ＿＿＿(＿＿) 小さい(0.97g/㎤) :水の密度よりも ＿＿ 低い
② 柔らかい。ナイフでも切れる。
③ 融点が ＿＿＿(＿＿) 　低い(98℃) :水の沸点より ＿＿ 低い
　(注)Kの沸点はNaよりも ＿＿＿＿ さらに低い
④ 常温の水と激しく反応する ＿＿＿＿ 2Na+2H₂O→2NaOH+H₂ (保管方法) ＿＿＿ 灯油 中
⑤ 熱したナトリウムは ＿＿＿ Cl(塩素) と激しく反応する。 ＿＿＿＿ 2Na+Cl₂→2NaCl
⑥ イオンは炎色反応を示す。(Na⁺： ＿＿＿ 黄 　K⁺： ＿＿＿ 赤紫 　Li⁺： ＿＿＿ 赤 )

2.NaOH

・工業的製法

＿＿＿ 塩化ナトリウム 水溶液の電気分解
[陰極]　 ＿＿＿ 2H₂O+2e⁻→OH⁻+H₂
[陽極]　 ＿＿＿ 2Cl⁻→Cl₂+2e⁻
　　⇨　[全体]　 ＿＿＿ 2NaCl+2H₂O→2NaOH+H₂+Cl₂

実験室的製法

＿＿＿ 酸化ナトリウム +水： ＿＿＿ Na₂O+H₂O→2NaOH

（補） ＿＿＿＿ 隔膜 法 と ＿＿＿＿ イオン交換膜 法
・ ＿＿＿＿ 隔膜 法：陽極側・陰極側ともに ＿＿＿＿ NaCl 水溶液を利用し、隔膜に ＿＿＿＿ 石綿 を使う。
（問題点） ＿＿＿＿ NaCl 水溶液中で ＿＿＿＿ NaOH が発生するため、NaOH 水溶液の濃度が ＿＿＿＿ 低い

・ ＿＿＿＿ イオン交換膜 法
陽極側溶液を ＿＿＿＿ NaCl 水溶液、陰極側水溶液を ＿＿＿＿ 薄い NaOH 水溶液を利用し、隔膜に、 ＿＿＿＿ 陽イオン交換膜 を使う。
＿＿＿＿ 陰極 側で発生する ＿＿＿＿ NaOH 水溶液の純度が ＿＿＿＿ 高い のが特徴。

[隔膜によって陽極側と陰極側を仕切る理由]
隔膜がないと生成した ＿＿＿＿ NaOH と ＿＿＿＿ Cl₂ で
＿＿＿＿ 2NaOH + Cl₂ → NaCl + NaClO + H₂O
と反応してしまい、NaOH の純度や収率が低下してしまう。

・性質

① 水によく溶け、水溶液は強塩基性を示す。
② ＿＿＿＿ 潮解 性を持つ。（ ＿＿＿＿ 潮解 性とは、空気中の水分を吸収して表面が濡れてくる現象）
③ ＿＿＿＿ 二酸化炭素 をよく吸収する。
　　 ＿＿＿＿ 2NaOH + CO₂ → Na₂CO₃ + H₂O
なお、さらに ＿＿＿＿ 二酸化炭素 を吸収させることもできる。
　　 ＿＿＿＿ Na₂CO₃ + CO₂ + H₂O → 2NaHCO₃
（注） ＿＿＿＿ KOH も同様に①〜③の性質を持つ。

3. Na₂CO₃

・工業的製法

アンモニアソーダ法
原料： ＿＿＿＿ NaCl（食塩） と ＿＿＿＿ CaCO₃（石灰石）
① ＿＿＿＿ 石灰石 を焼く
　　 ＿＿＿＿ CaCO₃ → CaO + CO₂
② ＿＿＿＿ NH₃（アンモニア） を先、 ＿＿＿＿ CO₂ を後で ＿＿＿＿ NaCl の飽和水溶液に吹き込み、溶解度の小さい ＿＿＿＿ NaHCO₃ を沈殿させ、 ＿＿＿＿ NH₄Cl 溶液を分離させる。
　　 ＿＿＿＿ NaCl + H₂O + NH₃ + CO₂ → NaHCO₃ + NH₄Cl
③ ②の ＿＿＿＿ NaHCO₃ を焼いて ＿＿＿＿ Na₂CO₃ を得る。
　　 ＿＿＿＿ 2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O
④ ①の ＿＿＿＿ CaO に水を作用させて ＿＿＿＿ Ca(OH)₂ とする。
　　 ＿＿＿＿ CaO + H₂O → Ca(OH)₂
⑤ ④でできた ＿＿＿＿ Ca(OH)₂ と②の ＿＿＿＿ NH₄Cl からアンモニアを取り出し、②のアンモニアに供給する。また、ここでできた ＿＿＿＿ CaCl₂(塩化カルシウム) は副産物の乾燥剤として回収する。
　　 ＿＿＿＿ 2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃ + 2H₂O
全体の式： ＿＿＿＿ 2NaCl + CaCO₃ → Na₂CO₃ + CaCl₂

・性質

① 水に溶け、水溶液は ＿＿＿＿ 比較的強い塩基 性を示す。
　　 ＿＿＿＿ CO₃²⁻ + H₂O ⇄ HCO₃⁻ + OH⁻
　補）炭酸水素ナトリウムも溶解度は小さいが水に溶ける。その際、 ＿＿＿＿ 弱い塩基性 となる。
　　 ＿＿＿＿ HCO₃⁻ ⇄ CO₂ + OH⁻
　なお、塩の分類としては ＿＿＿＿ 酸性塩 である。
② 酸に溶け、 ＿＿＿＿ 二酸化炭素 を発生する
　　 ＿＿＿＿ Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂
③ 熱分解しない。
炭酸ナトリウムは熱に対して安定で、高温にすると分解ではなく融解する。
　補）炭酸水素ナトリウムは熱分解して ＿＿＿＿ 二酸化炭素 を発生する
　　 ＿＿＿＿ 2NaHCO₃ → Na₂CO₃ + H₂O + CO₂
④ 水溶液から結晶を析出させると ＿＿＿＿ 十水和物 (Na₂CO₃・10H₂O) が得られる。
　補）炭酸水素ナトリウムは ＿＿＿＿ 風解 性を持ち ＿＿＿＿ 炭酸水素ナトリウム一水和物 になる。（ ＿＿＿＿ 風解 性とは空気中に放置すると水分を失っていく現象）

Caとその化合物

1. Ca

・製法

＿＿＿＿溶融塩電解
Na同様、イオン化傾向が ＿＿＿＿大きい ため、 ＿＿＿＿溶融塩電解 で作る。

・性質

① Caは ＿＿＿＿常温の水 と反応し、 ＿＿＿＿H₂ を発生する。
　　＿＿＿＿Ca + 2H₂O → Ca(OH)₂ + H₂
　補）Mgは ＿＿＿＿冷水 にはほとんど反応しないが、 ＿＿＿＿熱水 には反応する。

② 空気中で熱すると、激しく燃焼する。
　　＿＿＿＿2Ca + O₂ → 2CaO
　補）Mgも空気中で明るい光を出して燃焼する
　　＿＿＿＿2Mg + O₂ → 2MgO
　また、Mgは強い ＿＿＿＿還元 力を持つため、 ＿＿＿＿二酸化炭素 中でも燃焼する。
　　＿＿＿＿2Mg + CO₂ → 2MgO + C

③ イオンは炎色反応を示す。(Ca²⁺： ＿＿＿＿橙赤色 , Ba²⁺： ＿＿＿＿黄緑色 , Mg²⁺： ＿＿＿＿なし )

2. CaO（酸化カルシウム）

・製法

＿＿＿＿石灰石（CaCO₃） を焼くと酸化カルシウム（別名： ＿＿＿＿生石灰 ）が得られる。
　　＿＿＿＿CaCO₃ → CaO + CO₂

・性質

① 水と反応すると、大きな ＿＿＿＿発熱 を伴って激しく反応する。
　　＿＿＿＿CaO + H₂O → Ca(OH)₂

② 塩基性酸化物であり、酸と反応する。
　　＿＿＿＿CaO + 2HCl → CaCl₂ + H₂O

③ ＿＿＿＿コークス を作用させると ＿＿＿＿炭化カルシウム （カーバイド）が得られる。
　　＿＿＿＿CaO + 3C → CaC₂ + CO

　補）カーバイドは水を加えると ＿＿＿＿アセチレン ができる。
　　＿＿＿＿CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂

④ ＿＿＿＿乾燥剤 として用いられる。
酸化カルシウムに濃い水酸化ナトリウム水溶液を染み込ませ、焼いて粒状にしたものが ＿＿＿＿ソーダ石灰 である。

Ca(OH)₂（水酸化カルシウム）

・製法

酸化カルシウムの性質①を参照

・性質

① 水酸化カルシウム（別名： ＿＿＿＿消石灰 ）は水に少し溶け（別名： ＿＿＿＿石灰水 ）、強塩基性を示す。

② 水溶液は ＿＿＿＿二酸化炭素 を吸収させると白く濁り、沈澱が生じる（ ＿＿＿＿炭酸カルシウム の発生）
　　＿＿＿＿Ca(OH)₂ + CO₂ → CaCO₃ + H₂O

③ ②でさらに ＿＿＿＿二酸化炭素 を吸収させると、沈殿は溶解する。
　　＿＿＿＿CaCO₃ + H₂O + CO₂ → Ca(HCO₃)₂

　補）炭酸水素カルシウム水溶液を加熱すると再び沈殿が生じる。
　　＿＿＿＿Ca(HCO₃)₂ → CaCO₃ + H₂O + CO₂

④ 塩素を吸収させると、 ＿＿＿＿さらし粉 （漂白・殺菌剤）ができる。
　　＿＿＿＿Ca(OH)₂ + Cl₂ → CaCl(ClO)・H₂O

CaCO₃（炭酸カルシウム）

・性質

① 炭酸カルシウム（ ＿＿＿＿石灰石 ）は塩酸に溶け、 ＿＿＿＿二酸化炭素 を発生する。
　　＿＿＿＿CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂

② 熱分解する。
　　＿＿＿＿CaCO₃ → CaO + CO₂

③ 水に難溶である。

CaSO₄（硫酸カルシウム）

・性質

① 天然には ＿＿＿＿石膏（CaSO₄・2H₂O） として産出する。

② 水に難溶である。（補：MgSO₄は溶ける）

③ 石膏を加熱した ＿＿＿＿焼き石膏（CaSO₄・1/2H₂O） に水を加えて練り、放置すると発熱しながら、石膏に戻り硬化する。これがギプスや塑像に利用される。
　　＿＿＿＿CaSO₄・1/2H₂O + 3/2H₂O → CaSO₄・2H₂O

Alとその化合物

1.Al

・製法

① ＿＿＿＿ボーキサイト（Al₂O₃・nH₂O）を ＿＿＿＿濃い水酸化ナトリウム水溶液に入れて加熱して溶かして不純物（他の金属酸化物は ＿＿＿＿両性酸化物でないものが多く、 ＿＿＿＿NaOH(aq)には溶けずに沈澱する）を取り除く。
　　＿＿＿＿Al₂O₃ + 2NaOH + 3H₂O → 2Na[Al(OH)₄]

② の濾液に大量の水を加えると ＿＿＿＿Al(OH)₃が沈澱する。

③ ②の沈殿物を強熱すると純粋な ＿＿＿＿酸化アルミニウム（アルミナ）が得られる。
　　＿＿＿＿2Al(OH)₃ → Al₂O₃ + 3H₂O

④ ③で得られた ＿＿＿＿アルミナを融解させる。その際に、融点が約2050℃であるため、融点を下げるために ＿＿＿＿氷晶石（Na₃AlF₆）を加えて、融点を約1000℃ぐらいまで下げておく。
　　＿＿＿＿Al₂O₃ → 2Al³⁺ + 3O²⁻

⑤ ④の融解液を電気分解（溶融塩電解）する。
　　[陰極]　＿＿＿＿Al³⁺ + 3e⁻ → Al
　　[陽極]　＿＿＿＿C + O²⁻ → CO + 2e⁻
　　　　　または
　　＿＿＿＿C + 2O²⁻ → CO₂ + 4e⁻

補）Alの精錬において、他の金属の精錬よりも大量の電気を使う理由
・ ＿＿＿＿融解させるのに電気エネルギーが必要
・ Alのイオン価数が ＿＿＿＿3価（一般の金属は ＿＿＿＿2価が主流）であるため、 ＿＿＿＿1.5倍の電気が必要
・ 原子量が比較的 ＿＿＿＿小さい（27）

・性質

① 密度が ＿＿＿＿2.7 g·cm⁻³ で ＿＿＿＿軽い。このため、 ＿＿＿＿銅・マグネシウム・マンガンとの軽合金として ＿＿＿＿ジュラルミンが有名。

② 空気中で表面に緻密な酸化物 ＿＿＿＿Al₂O₃（酸化被膜）を作り、内部を保護して安定する。
アルミニウム製品の表面を人工的に酸化させた製品を ＿＿＿＿アルマイトといい、 ＿＿＿＿耐食性、 ＿＿＿＿絶縁性に優れ着色もできる。

③ ＿＿＿＿高温の水蒸気と反応する。
　　＿＿＿＿2Al + 6HCl → 2AlCl₃ + 3H₂

④ さんにも延期にも溶け水素を発生させる ＿＿＿＿両性金属である。
　　＿＿＿＿2Al + 6HCl → 2AlCl₃ + 3H₂
　　＿＿＿＿2Al + 3NaOH + 6H₂O → 2Na[Al(OH)₄] + 3H₂

⑤ ＿＿＿＿不動態形成物質である。そのため ＿＿＿＿濃硫酸・ ＿＿＿＿濃硝酸は表面の酸化被膜の形成のため溶けにくい。

⑥ 酸素とは反応しやすい。
　　＿＿＿＿4Al + 3O₂ → 2Al₂O₃
補）アルミニウムの粉末と酸化鉄との混合物に点火すると、多量の発熱を伴って激しく反応し、鉄の単体が遊離する。
　　＿＿＿＿2Al + Fe₂O₃ → Al₂O₃ + 2Fe
また、還元されにくい金属酸化物でもアルミニウム粉末との混合物の点火で金属単体が遊離する。この方法を ＿＿＿＿テルミット法という。
　　＿＿＿＿2Al + Cr₂O₃ → Al₂O₃ + 2Cr

Al₂O₃（酸化アルミニウム）

・性質

① 水に溶けず融点が非常に高い安定した物質（別名： ＿＿＿＿アルミナ）である。

② ＿＿＿＿両性酸化物であるため、酸にも塩基にも溶解する。
　　＿＿＿＿Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O
　　＿＿＿＿Al₂O₃ + 3NaOH + 3H₂O → 2Na[Al(OH)₄]

③ 酸化アルミニウムの結晶は ＿＿＿＿ダイヤモンドに次ぐ硬さを持ち、 ＿＿＿＿ルビー（+Cr）、 ＿＿＿＿サファイア（+Fe,Ti）の主成分である。

Al(OH)₃（水酸化アルミニウム）

・性質

① アルミニウムイオンに塩基性水溶液を加えるとできる ＿＿＿＿白色のゲル状沈澱である。

② ＿＿＿＿両性水酸化物であるため、酸にも塩基にも溶ける。
　　＿＿＿＿Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O
　　＿＿＿＿Al(OH)₃ + NaOH → Na[Al(OH)₄]

KAl(SO₄)₂·12H₂O（硫酸カリウムアルミニウム十二水和物）

・性質

① 硫酸カリウムと硫酸アルミニウムとの濃い水溶液を冷却させると ＿＿＿＿無色の ＿＿＿＿正八面体の結晶が得られる。これが硫酸カリウムアルミニウム十二水和物で別名 ＿＿＿＿ミョウバン（カリウムミョウバン）と呼ばれる。

② 二種類以上の塩がある一定の割合で結合した塩を ＿＿＿＿複塩といい、 ＿＿＿＿ミョウバンは複塩の代表例である。 ＿＿＿＿カリウムミョウバンの他にも ＿＿＿＿アンモニウムミョウバン（AlNH₄(SO₄)₂·12H₂O）や、 ＿＿＿＿鉄ミョウバン（FeNH₄(SO₄)₂·12H₂O）などもある。

③ 水溶液はすべて酸性を示す。

Znとその化合物

1.Zn

・性質

① ＿＿＿＿典型元素（13族〜）に近い遷移元素（12族）である。
② 合金やメッキにも用いられる。
　　＿＿＿＿黄銅：銅と亜鉛の合金

③ 両性元素である。
　　＿＿＿＿Zn＋2HCl → ZnCl₂＋H₂
　　＿＿＿＿Zn＋2NaOH＋2H₂O → Na₂[Zn(OH)₄]＋H₂

2.ZnO（酸化亜鉛）

・性質

① 亜鉛を燃焼させると得られる白色粉末である。
② 白色の顔料として、塗料や絵の具にも利用される。
③ 両性酸化物である。
　　＿＿＿＿ZnO＋2HCl → ZnCl₂＋H₂O
　　＿＿＿＿ZnO＋2NaOH＋H₂O → Na₂[Zn(OH)₄]

3.Zn(OH)₂（水酸化亜鉛）

・性質

① 亜鉛イオンに塩基の水溶液を加えると、 ＿＿＿＿白 色ゲル状沈澱として生成する。
② 両性水酸化物である。
　　＿＿＿＿Zn(OH)₂＋2HCl → ZnCl₂＋2H₂O
　　＿＿＿＿Zn(OH)₂＋2NaOH → Na₂[Zn(OH)₄]
③ アンモニア水に溶解し、 ＿＿＿＿無色のテトラアンミン亜鉛(II)イオン（正四面体） ができる。

4.ZnS（硫化亜鉛）

・性質

② 亜鉛イオンを含む ＿＿＿＿塩基性または中性 の水溶液に硫化水素H₂Sを吹き込むと沈澱する。
　　＿＿＿＿Zn²⁺＋S²⁻ → ZnS

Feとその化合物

1.Fe

・製法

① 鉄鉱石を ＿＿＿＿銑鉄 にする。
主な鉄鉱石：赤鉄鉱（化学式： ＿＿＿＿Fe₂O₃ ）・磁鉄鉱（化学式： ＿＿＿＿Fe₃O₄ ）
他に必要な原料： ＿＿＿＿コークス ・ ＿＿＿＿石灰石
・溶鉱炉に鉄鉱石・原料を入れ、下部から熱風を送り込むと、 ＿＿＿＿コークス が燃焼する。
　　＿＿＿＿2C＋O₂ → 2CO
・できた ＿＿＿＿一酸化炭素 が鉄鉱石を還元する。
　　＿＿＿＿Fe₂O₃＋3CO → 2Fe＋3CO₂
⇨ここで得られた鉄は ＿＿＿＿銑鉄 と呼ばれ、炭素含有量約4％の固くて脆い鉄で、 ＿＿＿＿鋳物 に利用される。

② ＿＿＿＿銑鉄 から ＿＿＿＿鋼 にする。
＿＿＿＿銑鉄 を転炉に移し、炉を回転させながら酸素を吹き込んで不純物を取り除く。ここで得られた鉄は ＿＿＿＿鋼 と呼ばれ、炭素含有量は0.02％〜2％まで減少しており、弾力性に富んだ強靭な ＿＿＿＿鋼材 となる。

③ 不純物の除去
鉄鉱石中の不純物であるSiO₂やAl₂O₃などは、石灰石と反応し銑鉄の上に浮かぶ ＿＿＿＿スラグ となって取り除かれる。

・性質

① 希硫酸に溶け ＿＿＿＿水素 を発生する。
　　＿＿＿＿Fe＋2HCl → FeCl₂＋H₂

＿＿＿＿FeCl₃（塩化鉄(III)） を生成するには、 ＿＿＿＿塩化鉄(II) 水溶液を ＿＿＿＿塩素 で酸化して得る。
　　＿＿＿＿2FeCl₂＋Cl₂ → 2FeCl₃
補） ＿＿＿＿塩化鉄(II) 水溶液は ＿＿＿＿淡緑 色、 ＿＿＿＿塩化鉄(III) 水溶液は ＿＿＿＿黄褐 色。
＿＿＿＿塩化鉄(II) 水溶液を濃縮すると、 ＿＿＿＿潮解 性を持つ ＿＿＿＿FeCl₃・6H₂O の結晶ができる。

② 希硫酸にも溶け ＿＿＿＿水素 を発生する。
　　＿＿＿＿Fe＋H₂SO₄ → FeSO₄＋H₂
補） ＿＿＿＿硫酸鉄(II)水溶液 は ＿＿＿＿淡緑 色で、濃縮すると ＿＿＿＿風解 性を持つ ＿＿＿＿FeSO₄・7H₂O の結晶ができる。

③ ＿＿＿＿濃硝酸 には ＿＿＿＿不動態 を形成するため溶けない。

④ 鉄に ＿＿＿＿クロム ・ ＿＿＿＿ニッケル を混ぜて作られる ＿＿＿＿ステンレス鋼 は、錆びにくい合金である。

2.酸化鉄

① 酸化鉄(III)：
＿＿＿＿赤褐 色で湿った空気で生じる ＿＿＿＿赤さび の主成分である。ベンガラとも呼ばれ赤色系顔料としても用いられる。水酸化鉄(III)を加熱しても生じる。
　　＿＿＿＿2Fe(OH)₃ → Fe₂O₃＋3H₂O

② 四酸化三鉄：
＿＿＿＿黒 色で ＿＿＿＿黒さび の主成分である。酸化鉄(III)を還元すると生じる。
　　＿＿＿＿3Fe₂O₃＋CO → 2Fe₃O₄＋CO₂

3.鉄イオンの反応

① アンモニア水または水酸化ナトリウム水溶液との反応
　　＿＿＿＿Fe²⁺＋2OH⁻ → Fe(OH)₂ （Fe²⁺： ＿＿＿＿淡緑 色　Fe(OH)₂： ＿＿＿＿緑白 色沈澱）
　　＿＿＿＿Fe³⁺＋3OH⁻ → Fe(OH)₃ （Fe³⁺： ＿＿＿＿黄褐 色　Fe(OH)₃： ＿＿＿＿赤褐 色沈澱）
補）水酸化鉄(II)は水溶液中の酸素によって酸化され水酸化鉄(III)になる。
　　＿＿＿＿4Fe(OH)₂＋O₂＋2H₂O → 4Fe(OH)₃

② ヘキサシアノ鉄(III)酸カリウムとの反応
化学式：K₃[Fe(CN)₆] 結晶では ＿＿＿＿暗赤 色、水溶液では ＿＿＿＿黄 色（ ＿＿＿＿正八面体 ）有毒。
⇨ ＿＿＿＿鉄(II)イオン を含む水溶液に反応： ＿＿＿＿濃青 色（ ＿＿＿＿ターンブル・ブルー ）沈澱

③ ヘキサシアノ鉄(II)酸カリウムとの反応
化学式：K₄[Fe(CN)₆] 結晶では三水和物で ＿＿＿＿黄 色、水溶液では ＿＿＿＿淡黄 色（ ＿＿＿＿正八面体 ）無毒。
⇨ ＿＿＿＿鉄(III)イオン を含む水溶液に反応： ＿＿＿＿濃青 色（ ＿＿＿＿プルシャン・ブルー ）沈澱

④ チオシアン酸カリウムとの反応
化学式：KSCN ＿＿＿＿潮解 性のある ＿＿＿＿無色 結晶。
⇨ ＿＿＿＿鉄(III)イオン を含む水溶液に反応： ＿＿＿＿血赤 色の溶液

1.Cu

・製法

① 黄鉄鉱から硫化銅（Ⅰ）にする
　黄鉄鉱をコークスやケイ砂などと混ぜて溶鉱炉に入れて硫化銅（Ⅰ）にする。
　　＿＿＿＿2CuFeS + 4O₂ + 2SiO₂ → Cu₂S + FeSiO₃ + 3SO₂

　硫化銅（Ⅰ）を転炉に入れ、銅の単体（粗銅）を得る。
　　＿＿＿＿Cu₂S + O₂ → 2Cu + SO₂
③ 粗銅から純銅にする
　陽極に＿＿＿＿粗銅、陰極に 　＿＿＿＿純銅をつけた 　＿＿＿＿電解精錬で純銅が得られる。
　　[陽極] Cu（粗） → Cu²⁺ + 2e⁻
　　[陰極] Cu²⁺ + 2e⁻ → Cu（純）
④ 不純物の除去
・銅よりイオン化傾向の＿＿＿＿大きい金属
　　イオンとなって溶液中に溶け出すが、陰極で単体で析出されない（銅のほうが析出）
・銅よりイオン化傾向の＿＿＿＿小さい金属
　　イオン化することなく、陽極の下に落ちてたまる（ 　　＿＿＿＿陽極泥）

・性質

① ＿＿＿＿赤色。展性・延性・電気伝導性に優れている。
② 希酸（希塩酸や希硫酸）には溶けない。
③ ＿＿＿＿酸化力を持つ酸には溶ける。
　熱濃硫酸：＿＿＿＿Cu + 2H₂SO₄ → CuSO₄ + SO₂ + 2H₂O　（⇨ ＿＿＿＿二酸化硫黄 の発生）
　濃硝酸　：＿＿＿＿Cu + 4HNO₃ → Cu(NO₃)₂ + 2NO₂ + 2H₂O　（⇨ ＿＿＿＿二酸化窒素 の発生）
　希硝酸　：＿＿＿＿3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 2NO + 4H₂O　（⇨ ＿＿＿＿一酸化窒素 の発生）

2.Cuの化合物

① 酸化銅（Ⅰ）：Cu₂O（＿＿＿＿赤色）
② 酸化銅（Ⅱ）：CuO（＿＿＿＿黒色）　水酸化銅（Ⅱ）の加熱で得られる。
③ 硫酸銅（Ⅱ）：CuSO₄
　　無水硫酸銅（Ⅱ）は＿＿＿＿白色。
　　硫酸銅水溶液から析出する結晶である 　　＿＿＿＿硫酸銅五水和物（CuSO₄·5H₂O）は 　　＿＿＿＿青色。
　　⇨ 無水硫酸銅（Ⅱ）は 　　＿＿＿＿水分 検出に利用される。

3.Cuの合金

① 黄銅（真鍮）：銅と＿＿＿＿亜鉛　　（用途）楽器
② 青銅（ブロンズ）：銅と＿＿＿＿スズ　　（用途）ブロンズ像
③ 白銅（キュプロニッケル）：銅と＿＿＿＿ニッケル　（用途）貨幣

4.Cuイオンの反応

① 水酸化ナトリウム水溶液との反応
　　＿＿＿＿Cu²⁺ + 2OH⁻ → Cu(OH)₂　（Cu²⁺：＿＿＿＿青色　Cu(OH)₂：＿＿＿＿青白色沈澱）
② アンモニア水との反応
　　＿＿＿＿Cu²⁺ + 2OH⁻ → Cu(OH)₂
（過剰に加えると）
　　＿＿＿＿Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄]²⁺ + 2OH⁻

C（14族）とその化合物

1.炭素の単体

・炭素の同素体

① ダイヤモンド
　・ダイヤモンドより硬いものは存在しない。
　・ある1個の炭素原子を＿＿＿＿正四面体の中心に置くと、 　　＿＿＿＿4個の炭素原子が 　　＿＿＿＿正四面体の各頂点に位置するように配置し、 　　その全てが＿＿＿＿共有結合で結びついた結晶構造をもつ。

② 黒鉛（グラファイト）
　・炭素原子が＿＿＿＿正六角形状に 　　次々と共有結合した網目状の平面構造（層）が、幾重にも重なり合った結晶構造を持つ。
　　層同士の間は、＿＿＿＿弱い分子間力でつながっているため、 　　一層一層が薄く剥がれやすい。
　・主に炭素原子の＿＿＿＿4個の価電子のうち 　　＿＿＿＿3個の働きによって層を作るため、 　　残りの＿＿＿＿1個の価電子は自由電子として移動し、 　　層内方向に＿＿＿＿電気伝導性を持つ。

2.炭素の酸化物

・CO（一酸化炭素）

① 製法
　工業的製法：空気中での不完全燃焼　 　＿＿＿＿C+O₂→2CO
　高温の炭素の単体に二酸化炭素が触れても発生　 　＿＿＿＿C+CO₂→2CO
　実験室的製法：＿＿＿＿蟻酸を濃硫酸で脱水　 　＿＿＿＿HCOOH→H₂O+CO

② 性質
・無色無臭
・水に溶けにくい
・極めて有毒
　血液中の＿＿＿＿ヘモグロビンと結合しやすく、 　＿＿＿＿ヘモグロビンが酸素と結合するのを妨げる。
・燃えて二酸化炭素になる
　空気中で点火すると青白い炎を上げ、二酸化炭素となる。　 　＿＿＿＿2CO+O₂→2CO₂
・高温で＿＿＿＿強い還元性を持つ
　高温で金属の酸化物を＿＿＿＿還元し金属単体を得る。
　非金属でも反応し、高温の水蒸気を＿＿＿＿還元して 　＿＿＿＿水素を発生させる。　 　＿＿＿＿H₂O+CO→H₂+CO₂

CO₂（二酸化炭素）

① 製法
　工業的製法：石灰石を強熱する　 　＿＿＿＿CaCO₃→CaO+CO₂
　実験室的製法：石灰石に希塩酸　 　＿＿＿＿CaCO₃+2HCl→CaCl₂+H₂O+CO₂

② 性質
・無色無臭
・水にいくらか溶ける（空気よりも＿＿＿＿重いので 　＿＿＿＿下方置換で捕集）
・毒性はない
・水酸化ナトリウム水溶液やソーダ石灰に吸収される　 　＿＿＿＿2NaOH+CO₂→Na₂CO₃+H₂O
・石灰水にも吸収され白濁する　 　＿＿＿＿Ca(OH)₂+CO₂→CaCO₃+H₂O
・二酸化炭素を含む気体の乾燥には、 　＿＿＿＿濃硫酸などの 　＿＿＿＿酸性乾燥剤が適当
・＿＿＿＿直線型の 　＿＿＿＿無極性分子
・＿＿＿＿31℃以下で 　＿＿＿＿圧力を加えると液体になる。
　⇨液体CO₂を大気中に噴出させると、 　＿＿＿＿蒸発熱により急激に温度が 　＿＿＿＿下がり、一部が ＿＿＿＿固体となる。 　これを集めてプレスしたものがドライアイスとなる。

Si（14族）とその化合物

1.ケイ素の単体

① 製法
　二酸化ケイ素を電気炉中で融解して、コークスで還元する　 ＿＿＿＿SiO₂+2C→Si+2CO

② 性質
・ケイ素の単体は天然には存在しない。
・ケイ素の元素は、地殻中に ＿＿＿＿酸素 という元素の次に多く存在する。
・ケイ素の単体は、 ＿＿＿＿ダイヤモンド と同じく ＿＿＿＿正四面体 構造を持つ、 ＿＿＿＿共有 結合の結晶である。
・融点が ＿＿＿＿高く 、硬いが脆い
・金属光沢を持ち、金属と非金属の中間の電気伝導性を持つため、 ＿＿＿＿半導体 の材料に用いられる。

2.ケイ素の化合物

・SiO₂（二酸化ケイ素）

① ＿＿＿＿石英 ・ ＿＿＿＿水晶 ・ケイ砂など天然に多く存在する。
② ＿＿＿＿共有 結合の結晶であり、融点は ＿＿＿＿高 く硬い。
③ ＿＿＿＿フッ化水素酸 に反応し、 ＿＿＿＿ヘキサフルオロケイ酸 となって溶ける。
　　＿＿＿＿SiO₂+6HF→H₂SiF₆+2H₂O

④ ＿＿＿＿ガラス の主成分である。
　（ ＿＿＿＿ガラス の特徴）
　・ ＿＿＿＿非晶質 （決まった融点がなく配列が不規則な固体）
　・着色可能で熱可塑性

・Na₂SiO₃（ケイ酸ナトリウム）

① 二酸化ケイ素の粉末を水酸化ナトリウムや炭酸ナトリウムと共に高温で ＿＿＿＿融解 すると得られる。
　　＿＿＿＿SiO₂+2NaOH→Na₂SiO₃+H₂O
　　＿＿＿＿SiO₂+Na₂CO₃→Na₂SiO₃+CO₂

② ガラス状の固体である。
③ 水を加えて長時間煮沸すると粘性の大きい無色透明の液体となる。 この液体を ＿＿＿＿水ガラス という。

・H₂SiO₃（ケイ酸）

① ＿＿＿＿水ガラス の水溶液に塩酸を加えると、ゲル状の白い半透明のケイ酸が沈澱する。
　　＿＿＿＿Na₂SiO₃+2HCl→H₂SiO₃+2NaCl

② ケイ酸を乾燥させたものが ＿＿＿＿シリカゲル である。
　⇨ ＿＿＿＿シリカゲル は ＿＿＿＿多孔 質で、表面に親水性のヒドロキシ基を多く持つため、水蒸気を吸着する力が強く、吸湿・乾燥剤に用いられる。
　なお、 ＿＿＿＿塩化コバルト を含むシリカゲルは、コバルト(Ⅱ)イオンのため、乾燥時は ＿＿＿＿青 色、吸湿後は ＿＿＿＿桃 色となる。

N（15族）とその化合物

1.N₂

① 製法
・工業的製法：＿＿＿＿液体空気の分留
・実験室的製法：＿＿＿＿亜硝酸アンモニウムの熱分解　＿＿＿＿NH₄NO₂ → N₂ + 2H₂O￥
② 性質
・空気中に約78％存在する。
・常温常圧で＿＿＿＿貴ガスについで不活発な気体である。
・高温状態では、酸素と化合して窒素酸化物を生じるなど、様々な元素と反応する。
・液体窒素は＿＿＿＿冷却剤として用いられる。

2.窒素の酸化物

・NO（一酸化窒素）

① 製法
・工業的製法：オストワルト法の第1段階目　＿＿＿＿4NH₃ + 5O₂ → 4NO + 6H₂O￥
・実験室的製法：銅＋希酸　＿＿＿＿3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 2NO + 4H₂O￥
② 性質
・＿＿＿＿無色 無臭の気体
・水に＿＿＿＿溶けにくい（捕集方法：水上置換）
・空気中に直ちに酸化され、二酸化窒素になる。　＿＿＿＿2NO + O₂ → 2NO₂￥

・NO₂（二酸化窒素）

① 製法
・工業的製法：オストワルトの第2段階目まで
・実験室的製法：銅＋濃硝酸　＿＿＿＿Cu + 4HNO₃ → Cu(NO₃)₂ + 2NO₂ + 2H₂O￥
② 性質
・＿＿＿＿赤褐色 刺激臭の気体
・有毒
・水に＿＿＿＿溶けやすい、空気より＿＿＿＿重い（捕集方法：下方置換））
・常温では一部が＿＿＿＿四酸化二窒素（無色）に変化する。　＿＿＿＿2NO₂ ⇄ N₂O₄￥
・水に溶けて硝酸を生成する。
（高温の水）＿＿＿＿3NO₂ + H₂O → 2HNO₃ + NO￥
（低温の水）＿＿＿＿2NO₂ + H₂O → HNO₃ + HNO₂￥

3.その他の窒素化合物

・NH₃（アンモニア）

① 製法
・工業的製法：＿＿＿＿ハーバー＝ボッシュ法　＿＿＿＿N₂ + 3H₂ ⇄ 2NH₃￥
1）触媒：＿＿＿＿四酸化三鉄（Fe₃O₄）を主成分とする鉄触媒
（触媒を使う理由）
＿＿＿＿発熱反応であることから、温度を上げるとアンモニアの生成量が落ちる。反応速度の確保のため触媒が必要。
2）反応条件：＿＿＿＿高温（約 ＿＿＿＿500℃） ＿＿＿＿高圧（約300気圧）
3）分離方法：実反応の窒素や水素と分離するため、冷却して凝縮して液体として取り出す。
・実験室的製法：＿＿＿＿アンモニウム塩＋＿＿＿＿強塩基（加熱）
例）塩化アンモニウム＋水酸化カルシウム　＿＿＿＿2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃ + 2H₂O￥
（この時、共に出てくる水蒸気を取り除くため乾燥剤が必要：＿＿＿＿ソーダ石灰が適当）

② 性質
・＿＿＿＿無色 刺激臭の気体
・水に＿＿＿＿極めて溶けやすく、空気より＿＿＿＿軽い（捕集方法：水上置換））
・液体アンモニアが蒸発する時の蒸発熱が大きく、冷媒として冷凍機などに利用。
・塩化水素が空気中で触れると、塩化アンモニウムの微細な結晶が生成し、白煙を生じる。　＿＿＿＿NH₃ + HCl → NH₄Cl￥
・二酸化炭素を反応させると＿＿＿＿尿素を合成できる。　＿＿＿＿2NH₃ + CO₂ → CO(NH₂)₂ + H₂O￥

・HNO₃（硝酸）

① 製法
・工業的製法：＿＿＿＿オストワルト法￥
第1段階：アンモニアと空気の混合気体を＿＿＿＿800℃に加熱した＿＿＿＿白金網（触媒）に接触させる。
　　＿＿＿＿4NH₃ + 5O₂ → 4NO + 6H₂O￥
第2段階：冷却後、一酸化窒素を空気中の酸素と反応させて二酸化窒素を作る。
　　＿＿＿＿2NO + O₂ → 2NO₂￥
第3段階：二酸化窒素を＿＿＿＿温水に吸収させて硝酸にする。
　　＿＿＿＿3NO₂ + H₂O → 2HNO₃ + NO￥
（全体の式）＿＿＿＿NH₃ + 2O₂ → HNO₃ + H₂O￥
・実験室的製法：硝酸ナトリウム＋濃硫酸　＿＿＿＿NaNO₃ + H₂SO₄ → NaHSO₄ + HNO₃￥

② 性質
・＿＿＿＿酸化力￥が強い
1）イオン化傾向の＿＿＿＿小さい金属（Cu, Hg, Agなど）でも溶かすことができる。
2）＿＿＿＿Ag, Fe, Niに濃硝酸を加えても、表面に緻密な酸化被膜が生じ＿＿＿＿不動態が形成され、それ以上は反応しない。
・揮発性の酸
　硝酸の縁に不揮発性の酸である濃硫酸を加えると、硝酸が遊離する（実験室的製法）
・濃硝酸は光や熱で分解しやすい　＿＿＿＿4HNO₃ → 2H₂O + O₂ + 4NO₂￥
　⇨＿＿＿＿褐色￥瓶に入れて＿＿＿＿冷暗￥所に保存
（なお古くなった硝酸は分解によって生成した二酸化窒素の影響で＿＿＿＿淡黄￥色をおびている。）

黄リン

・二硫化炭素に＿＿＿＿溶解する￥。

赤リン

① 製法：黄リンに対して、＿＿＿＿空気を絶って約250℃に加熱する￥と赤リンになる。
② 性質

・二硫化炭素に＿＿＿＿溶解しない￥。

5.リンの化合物

・P₄O₁₀（十酸化四リン）

① 製法：リンの燃焼　赤リン＿＿＿＿4P + 5O₂ → P₄O₁₀￥　黄リン＿＿＿＿P₄ + 5O₂ → P₄O₁₀￥
② 性質
　・白色の固体
　・＿＿＿＿吸湿￥性が強く、＿＿＿＿乾燥￥剤に利用

・H₃PO₄（リン酸）

① 製法：十酸化四リンを水に溶かして加熱する。　＿＿＿＿P₄O₁₀ + 6H₂O → 4H₃PO₄￥
② 性質
・＿＿＿＿白￥色の固体
・水溶液は＿＿＿＿中程度の強さの酸性￥を示す。

・Ca₃(PO₄)₂（リン酸カルシウム）

① 性質
・リン鉱石の主成分
・水に＿＿＿＿難溶￥
・動物の歯や骨の主成分
・硫酸との反応でできる＿＿＿＿リン酸二水素カルシウム￥と＿＿＿＿硫酸カルシウム￥の混合物は、＿＿＿＿過リン酸石灰￥と呼ばれ、肥料に用いられる。
　　＿＿＿＿Ca₃(PO₄)₂ + 2H₂SO₄ → Ca(H₂PO₄)₂ + 2CaSO₄￥
なおCa(H₂PO₄)₂は水溶性のため、植物は水と共に根から吸収できる。
