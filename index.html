Naとその化合物

1.Na

・製法

＿＿＿＿ 溶融塩電解(融解塩電解)
塩化ナトリウム・水酸化ナトリウムの固体をそのまま融解した液体を電気分解する。 　　　[陰極] ＿＿＿＿ Na⁺+e⁻→Na

・性質

① 密度が ＿＿＿(＿＿) 小さい(0.97g/㎤) :水の密度よりも ＿＿ 低い
② 柔らかい。ナイフでも切れる。
③ 融点が ＿＿＿(＿＿) 　低い(98℃) :水の沸点より ＿＿ 低い
　(注)Kの沸点はNaよりも ＿＿＿＿ さらに低い
④ 常温の水と激しく反応する ＿＿＿＿ 2Na+2H₂O→2NaOH+H₂ (保管方法) ＿＿＿ 灯油 中
⑤ 熱したナトリウムは ＿＿＿ Cl(塩素) と激しく反応する。 ＿＿＿＿ 2Na+Cl₂→2NaCl
⑥ イオンは炎色反応を示す。(Na⁺： ＿＿＿ 黄 　K⁺： ＿＿＿ 赤紫 　Li⁺： ＿＿＿ 赤 )

2.NaOH

・工業的製法

＿＿＿ 塩化ナトリウム 水溶液の電気分解
[陰極]　 ＿＿＿ 2H₂O+2e⁻→OH⁻+H₂
[陽極]　 ＿＿＿ 2Cl⁻→Cl₂+2e⁻
　　⇨　[全体]　 ＿＿＿ 2NaCl+2H₂O→2NaOH+H₂+Cl₂

実験室的製法

＿＿＿ 酸化ナトリウム +水： ＿＿＿ Na₂O+H₂O→2NaOH

（補） ＿＿＿＿ 隔膜 法 と ＿＿＿＿ イオン交換膜 法
・ ＿＿＿＿ 隔膜 法：陽極側・陰極側ともに ＿＿＿＿ NaCl 水溶液を利用し、隔膜に ＿＿＿＿ 石綿 を使う。
（問題点） ＿＿＿＿ NaCl 水溶液中で ＿＿＿＿ NaOH が発生するため、NaOH 水溶液の濃度が ＿＿＿＿ 低い

・ ＿＿＿＿ イオン交換膜 法
陽極側溶液を ＿＿＿＿ NaCl 水溶液、陰極側水溶液を ＿＿＿＿ 薄い NaOH 水溶液を利用し、隔膜に、 ＿＿＿＿ 陽イオン交換膜 を使う。
＿＿＿＿ 陰極 側で発生する ＿＿＿＿ NaOH 水溶液の純度が ＿＿＿＿ 高い のが特徴。

[隔膜によって陽極側と陰極側を仕切る理由]
隔膜がないと生成した ＿＿＿＿ NaOH と ＿＿＿＿ Cl₂ で
＿＿＿＿ 2NaOH + Cl₂ → NaCl + NaClO + H₂O
と反応してしまい、NaOH の純度や収率が低下してしまう。

・性質

① 水によく溶け、水溶液は強塩基性を示す。
② ＿＿＿＿ 潮解 性を持つ。（ ＿＿＿＿ 潮解 性とは、空気中の水分を吸収して表面が濡れてくる現象）
③ ＿＿＿＿ 二酸化炭素 をよく吸収する。
　　 ＿＿＿＿ 2NaOH + CO₂ → Na₂CO₃ + H₂O
なお、さらに ＿＿＿＿ 二酸化炭素 を吸収させることもできる。
　　 ＿＿＿＿ Na₂CO₃ + CO₂ + H₂O → 2NaHCO₃
（注） ＿＿＿＿ KOH も同様に①〜③の性質を持つ。

3. Na₂CO₃

・工業的製法

アンモニアソーダ法
原料： ＿＿＿＿ NaCl（食塩） と ＿＿＿＿ CaCO₃（石灰石）
① ＿＿＿＿ 石灰石 を焼く
　　 ＿＿＿＿ CaCO₃ → CaO + CO₂
② ＿＿＿＿ NH₃（アンモニア） を先、 ＿＿＿＿ CO₂ を後で ＿＿＿＿ NaCl の飽和水溶液に吹き込み、溶解度の小さい ＿＿＿＿ NaHCO₃ を沈殿させ、 ＿＿＿＿ NH₄Cl 溶液を分離させる。
　　 ＿＿＿＿ NaCl + H₂O + NH₃ + CO₂ → NaHCO₃ + NH₄Cl
③ ②の ＿＿＿＿ NaHCO₃ を焼いて ＿＿＿＿ Na₂CO₃ を得る。
　　 ＿＿＿＿ 2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O
④ ①の ＿＿＿＿ CaO に水を作用させて ＿＿＿＿ Ca(OH)₂ とする。
　　 ＿＿＿＿ CaO + H₂O → Ca(OH)₂
⑤ ④でできた ＿＿＿＿ Ca(OH)₂ と②の ＿＿＿＿ NH₄Cl からアンモニアを取り出し、②のアンモニアに供給する。また、ここでできた ＿＿＿＿ CaCl₂(塩化カルシウム) は副産物の乾燥剤として回収する。
　　 ＿＿＿＿ 2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃ + 2H₂O
全体の式： ＿＿＿＿ 2NaCl + CaCO₃ → Na₂CO₃ + CaCl₂

・性質

① 水に溶け、水溶液は ＿＿＿＿ 比較的強い塩基 性を示す。
　　 ＿＿＿＿ CO₃²⁻ + H₂O ⇄ HCO₃⁻ + OH⁻
　補）炭酸水素ナトリウムも溶解度は小さいが水に溶ける。その際、 ＿＿＿＿ 弱い塩基性 となる。
　　 ＿＿＿＿ HCO₃⁻ ⇄ CO₂ + OH⁻
　なお、塩の分類としては ＿＿＿＿ 酸性塩 である。
② 酸に溶け、 ＿＿＿＿ 二酸化炭素 を発生する
　　 ＿＿＿＿ Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂
③ 熱分解しない。
炭酸ナトリウムは熱に対して安定で、高温にすると分解ではなく融解する。
　補）炭酸水素ナトリウムは熱分解して ＿＿＿＿ 二酸化炭素 を発生する
　　 ＿＿＿＿ 2NaHCO₃ → Na₂CO₃ + H₂O + CO₂
④ 水溶液から結晶を析出させると ＿＿＿＿ 十水和物 (Na₂CO₃・10H₂O) が得られる。
　補）炭酸水素ナトリウムは ＿＿＿＿ 風解 性を持ち ＿＿＿＿ 炭酸水素ナトリウム一水和物 になる。（ ＿＿＿＿ 風解 性とは空気中に放置すると水分を失っていく現象）

⬅ ホームへ戻る

Caとその化合物

1. Ca

・製法

＿＿＿＿溶融塩電解
Na同様、イオン化傾向が ＿＿＿＿大きい ため、 ＿＿＿＿溶融塩電解 で作る。

・性質

① Caは ＿＿＿＿常温の水 と反応し、 ＿＿＿＿H₂ を発生する。
　　＿＿＿＿Ca + 2H₂O → Ca(OH)₂ + H₂
　補）Mgは ＿＿＿＿冷水 にはほとんど反応しないが、 ＿＿＿＿熱水 には反応する。

② 空気中で熱すると、激しく燃焼する。
　　＿＿＿＿2Ca + O₂ → 2CaO
　補）Mgも空気中で明るい光を出して燃焼する
　　＿＿＿＿2Mg + O₂ → 2MgO
　また、Mgは強い ＿＿＿＿還元 力を持つため、 ＿＿＿＿二酸化炭素 中でも燃焼する。
　　＿＿＿＿2Mg + CO₂ → 2MgO + C

③ イオンは炎色反応を示す。(Ca²⁺： ＿＿＿＿橙赤色 , Ba²⁺： ＿＿＿＿黄緑色 , Mg²⁺： ＿＿＿＿なし )

2. CaO（酸化カルシウム）

・製法

＿＿＿＿石灰石（CaCO₃） を焼くと酸化カルシウム（別名： ＿＿＿＿生石灰 ）が得られる。
　　＿＿＿＿CaCO₃ → CaO + CO₂

・性質

① 水と反応すると、大きな ＿＿＿＿発熱 を伴って激しく反応する。
　　＿＿＿＿CaO + H₂O → Ca(OH)₂

② 塩基性酸化物であり、酸と反応する。
　　＿＿＿＿CaO + 2HCl → CaCl₂ + H₂O

③ ＿＿＿＿コークス を作用させると ＿＿＿＿炭化カルシウム （カーバイド）が得られる。
　　＿＿＿＿CaO + 3C → CaC₂ + CO

　補）カーバイドは水を加えると ＿＿＿＿アセチレン ができる。
　　＿＿＿＿CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂

④ ＿＿＿＿乾燥剤 として用いられる。
酸化カルシウムに濃い水酸化ナトリウム水溶液を染み込ませ、焼いて粒状にしたものが ＿＿＿＿ソーダ石灰 である。

Ca(OH)₂（水酸化カルシウム）

・製法

酸化カルシウムの性質①を参照

・性質

① 水酸化カルシウム（別名： ＿＿＿＿消石灰 ）は水に少し溶け（別名： ＿＿＿＿石灰水 ）、強塩基性を示す。

② 水溶液は ＿＿＿＿二酸化炭素 を吸収させると白く濁り、沈澱が生じる（ ＿＿＿＿炭酸カルシウム の発生）
　　＿＿＿＿Ca(OH)₂ + CO₂ → CaCO₃ + H₂O

③ ②でさらに ＿＿＿＿二酸化炭素 を吸収させると、沈殿は溶解する。
　　＿＿＿＿CaCO₃ + H₂O + CO₂ → Ca(HCO₃)₂

　補）炭酸水素カルシウム水溶液を加熱すると再び沈殿が生じる。
　　＿＿＿＿Ca(HCO₃)₂ → CaCO₃ + H₂O + CO₂

④ 塩素を吸収させると、 ＿＿＿＿さらし粉 （漂白・殺菌剤）ができる。
　　＿＿＿＿Ca(OH)₂ + Cl₂ → CaCl(ClO)・H₂O

CaCO₃（炭酸カルシウム）

・性質

① 炭酸カルシウム（ ＿＿＿＿石灰石 ）は塩酸に溶け、 ＿＿＿＿二酸化炭素 を発生する。
　　＿＿＿＿CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂

② 熱分解する。
　　＿＿＿＿CaCO₃ → CaO + CO₂

③ 水に難溶である。

CaSO₄（硫酸カルシウム）

・性質

① 天然には ＿＿＿＿石膏（CaSO₄・2H₂O） として産出する。

② 水に難溶である。（補：MgSO₄は溶ける）

③ 石膏を加熱した ＿＿＿＿焼き石膏（CaSO₄・1/2H₂O） に水を加えて練り、放置すると発熱しながら、石膏に戻り硬化する。これがギプスや塑像に利用される。
　　＿＿＿＿CaSO₄・1/2H₂O + 3/2H₂O → CaSO₄・2H₂O

Alとその化合物

1.Al

・製法

① ＿＿＿＿ボーキサイト（Al₂O₃・nH₂O）を ＿＿＿＿濃い水酸化ナトリウム水溶液に入れて加熱して溶かして不純物（他の金属酸化物は ＿＿＿＿両性酸化物でないものが多く、 ＿＿＿＿NaOH(aq)には溶けずに沈澱する）を取り除く。
　　＿＿＿＿Al₂O₃ + 2NaOH + 3H₂O → 2Na[Al(OH)₄]

② の濾液に大量の水を加えると ＿＿＿＿Al(OH)₃が沈澱する。

③ ②の沈殿物を強熱すると純粋な ＿＿＿＿酸化アルミニウム（アルミナ）が得られる。
　　＿＿＿＿2Al(OH)₃ → Al₂O₃ + 3H₂O

④ ③で得られた ＿＿＿＿アルミナを融解させる。その際に、融点が約2050℃であるため、融点を下げるために ＿＿＿＿氷晶石（Na₃AlF₆）を加えて、融点を約1000℃ぐらいまで下げておく。
　　＿＿＿＿Al₂O₃ → 2Al³⁺ + 3O²⁻

⑤ ④の融解液を電気分解（溶融塩電解）する。
　　[陰極]　＿＿＿＿Al³⁺ + 3e⁻ → Al
　　[陽極]　＿＿＿＿C + O²⁻ → CO + 2e⁻
　　　　　または
　　＿＿＿＿C + 2O²⁻ → CO₂ + 4e⁻

補）Alの精錬において、他の金属の精錬よりも大量の電気を使う理由
・ ＿＿＿＿融解させるのに電気エネルギーが必要
・ Alのイオン価数が ＿＿＿＿3価（一般の金属は ＿＿＿＿2価が主流）であるため、 ＿＿＿＿1.5倍の電気が必要
・ 原子量が比較的 ＿＿＿＿小さい（27）

・性質

① 密度が ＿＿＿＿2.7 g·cm⁻³ で ＿＿＿＿軽い。このため、 ＿＿＿＿銅・マグネシウム・マンガンとの軽合金として ＿＿＿＿ジュラルミンが有名。

② 空気中で表面に緻密な酸化物 ＿＿＿＿Al₂O₃（酸化被膜）を作り、内部を保護して安定する。
アルミニウム製品の表面を人工的に酸化させた製品を ＿＿＿＿アルマイトといい、 ＿＿＿＿耐食性、 ＿＿＿＿絶縁性に優れ着色もできる。

③ ＿＿＿＿高温の水蒸気と反応する。
　　＿＿＿＿2Al + 6HCl → 2AlCl₃ + 3H₂

④ さんにも延期にも溶け水素を発生させる ＿＿＿＿両性金属である。
　　＿＿＿＿2Al + 6HCl → 2AlCl₃ + 3H₂
　　＿＿＿＿2Al + 3NaOH + 6H₂O → 2Na[Al(OH)₄] + 3H₂

⑤ ＿＿＿＿不動態形成物質である。そのため ＿＿＿＿濃硫酸・ ＿＿＿＿濃硝酸は表面の酸化被膜の形成のため溶けにくい。

⑥ 酸素とは反応しやすい。
　　＿＿＿＿4Al + 3O₂ → 2Al₂O₃
補）アルミニウムの粉末と酸化鉄との混合物に点火すると、多量の発熱を伴って激しく反応し、鉄の単体が遊離する。
　　＿＿＿＿2Al + Fe₂O₃ → Al₂O₃ + 2Fe
また、還元されにくい金属酸化物でもアルミニウム粉末との混合物の点火で金属単体が遊離する。この方法を ＿＿＿＿テルミット法という。
　　＿＿＿＿2Al + Cr₂O₃ → Al₂O₃ + 2Cr

Al₂O₃（酸化アルミニウム）

・性質

① 水に溶けず融点が非常に高い安定した物質（別名： ＿＿＿＿アルミナ）である。

② ＿＿＿＿両性酸化物であるため、酸にも塩基にも溶解する。
　　＿＿＿＿Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O
　　＿＿＿＿Al₂O₃ + 3NaOH + 3H₂O → 2Na[Al(OH)₄]

③ 酸化アルミニウムの結晶は ＿＿＿＿ダイヤモンドに次ぐ硬さを持ち、 ＿＿＿＿ルビー（+Cr）、 ＿＿＿＿サファイア（+Fe,Ti）の主成分である。

Al(OH)₃（水酸化アルミニウム）

・性質

① アルミニウムイオンに塩基性水溶液を加えるとできる ＿＿＿＿白色のゲル状沈澱である。

② ＿＿＿＿両性水酸化物であるため、酸にも塩基にも溶ける。
　　＿＿＿＿Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O
　　＿＿＿＿Al(OH)₃ + NaOH → Na[Al(OH)₄]

KAl(SO₄)₂·12H₂O（硫酸カリウムアルミニウム十二水和物）

・性質

① 硫酸カリウムと硫酸アルミニウムとの濃い水溶液を冷却させると ＿＿＿＿無色の ＿＿＿＿正八面体の結晶が得られる。これが硫酸カリウムアルミニウム十二水和物で別名 ＿＿＿＿ミョウバン（カリウムミョウバン）と呼ばれる。

② 二種類以上の塩がある一定の割合で結合した塩を ＿＿＿＿複塩といい、 ＿＿＿＿ミョウバンは複塩の代表例である。 ＿＿＿＿カリウムミョウバンの他にも ＿＿＿＿アンモニウムミョウバン（AlNH₄(SO₄)₂·12H₂O）や、 ＿＿＿＿鉄ミョウバン（FeNH₄(SO₄)₂·12H₂O）などもある。

③ 水溶液はすべて酸性を示す。
